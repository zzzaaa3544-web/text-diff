# Text Diff

Compare two texts side by side and highlight the differences between them.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Open Diff Tool** (or Skip).

**Step 3:** Paste the original text in the **left panel** (ORIGINAL).

**Step 4:** Paste the modified text in the **right panel** (MODIFIED).

**Step 5:** Click **Compare** (or it auto-compares as you type).

**Step 6:** The output shows differences:
- `Green lines with +` = added text
- `Red lines with -` = removed text
- `Grey lines` = unchanged text

**Step 7:** Click **Load Example** to see a sample comparison.

**Step 8:** The stats bar shows total added, removed, and unchanged lines.

## Features
- Side-by-side text comparison
- Line-by-line diff highlighting
- Added (green) / Removed (red) / Same (grey) color coding
- Auto-compare on typing
- Line count statistics
- Load example for demo
